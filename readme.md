# LC3 - Little Computer 3

This is a simple LC3 emulator written in C99

## Building

```
make
```

## Running

Note: This is only an emulator, and does not includes the compiler, so you need to compile your assembler

```
./lc3 example/hello.obj
Hello, WorldHALT
```
